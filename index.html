sobriquet |ˈsəʊbrɪkeɪ|

noun

a person's nickname. she was a vast and haughty person who answered to the sobriquet ‘Duchesse’.

ORIGIN mid 17th century: French, originally in the sense ‘tap under the chin’, of unknown origin.

Sendmail Sobriquet is a web-based app that allows you to manage a Sendmail alias database stored in a LDAP directory server. If you run your own Sendmail server and find yourself making changes to /etc/mail/aliases all the time, this tool can make your life easier. If you already have Sendmail configured to use a LDAP-based alias database, this tool can still make your life easier.
